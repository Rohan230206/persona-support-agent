# Multi-Factor Authentication (MFA) Configuration Guide

Multi-Factor Authentication adds a layer of security to user logins.

## Supported MFA Methods
- **Authenticator Apps (TOTP)**: Google Authenticator, Authy, Microsoft Authenticator.
- **SMS Passcodes**: Fallback method for verified mobile numbers.
- **Security Keys**: WebAuthn / FIDO2 security devices.

## Step-by-Step Setup
1. Log in to the User Dashboard and navigate to **Security Settings**.
2. Click **Enable Multi-Factor Authentication**.
3. Select **Authenticator App** (Recommended). The system will generate a QR code and a secret setup key.
4. Scan the QR code with your authenticator app.
5. Enter the 6-digit verification code generated by the app to confirm successful pairing.
6. Save the **MFA Backup Codes** in a secure offline location.

## MFA Recovery
If you lose access to your authenticator device:
- Use one of your 10 single-use Backup Codes. Each backup code is invalid after one use.
- If backup codes are lost, contact your organization's Administrator to reset your MFA configuration. Administrators can reset MFA via `Admin Panel > Users > MFA > Reset MFA`.
